LandHydrology.Simulations.Simulation — Type
struct Simulation <: AbstractSimulation

A simulation wraps an abstract LandHydrology model containing equation specifications and an instance of an integrator used for time integration of the discretized model PDE.

Fields

  • model

    A LandHydrology model

  • integrator

    a DiffEqBase.jl integrator used for time

  • callbacks

    user defined callback operations
